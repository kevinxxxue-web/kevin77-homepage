# Kevin Xue

An ultra-minimal personal homepage for Kevin Xue, aka Kevin77.

The visual direction is **Pacific Twilight**: a restrained Los Angeles coastal sunset, warm orange and dusty mauve, organic film texture, and nearly imperceptible motion across the sky, haze, ocean, and distant city lights.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
